## Contribute
1. `$ npm install -g hexo-cli`
2. `$ git clone https://github.com/TechBridgeHQ/techbridgehq.github.io.git`
3. 寫新文章 `$ hexo new post <title>`，開始使用 Markdown 撰寫文章
4. 重新 compile 一次 `$hexo g`
5. 在本地端(http://localhost:4000)觀看 `$hexo s`
6. Deploy `$hexo d`
